Adds bulk actions for hiding/showing Google feed products via WooCommerce Product Feed.

== Description ==

### Adds bulk action to WooCommerce Product Feed.

This plugin enables hidding/showing product via bulk actions.

Please notice that WooCommerce Product Feed must be installed and active.

* https://woocommerce.com/products/google-product-feed/


== Installation ==

1. Upload plugin files to your plugins folder, or install using WordPress' built-in Add New Plugin installer
1. Activate the plugin
1. Go to Produtcs
1. Enjoy bulk actions for hiding/showing products
